# Ad / business reference assets

Source material only — **nothing here is served by the site.** Vite only serves
`public/`, so these files are never bundled, fetched, or linked at runtime.

What it is: screenshots of the client's Google-Maps info panels plus downloaded
creatives, captured 2026-07-21. Address and phone values in
`src/data/haninBusinesses.ts` were transcribed by hand from these panels where
the business/panel pairing was confident.

Images the site actually renders live in `public/brand/hanin/` (showcase grid and
wing banners). If a creative here needs to go live, process it and add it there.
